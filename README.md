# WordPress Plugin 10Web Module
A module for handling 10Web functionality in the plugin or other integrations with the theme.

## Module Responsibilities

- Any integrations specific to the 10web wvc-theme.

## Critical Paths

- This module has no user interface, it just adds some connections constants or integrations with the theme.

## Installation

### 1. Add the Newfold Satis to your `composer.json`.

```bash
composer config repositories.newfold composer https://newfold-labs.github.io/satis
```

### 2. Require the `newfold-labs/wp-module-10web` package.

```bash
composer require newfold-labs/wp-module-10web
```
